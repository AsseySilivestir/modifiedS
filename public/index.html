What this replaces

The original Splannes app was a Next.js fullstack project. modifiedS rewrites only the backend — every Next.js route handler under /app/api/** becomes a Sua route registered in routes.b.

  • app/api/auth/{register,login,me}/route.ts → auth.b
  • app/api/roadmaps/.../route.ts → roadmaps.b
  • app/api/progress/.../route.ts → progress.b
  • app/api/notes/.../route.ts → notes.b
  • app/api/ai/{tutor,quiz}/route.ts → ai.b (rule-based, no API key)
  • prisma/schema.prisma → initDb() in db.b
  • prisma/seed.ts → seed.b (56 roadmaps)

Run it
